# Databases

## Focus areas for interviews:
- **PostgreSQL / Relational**: ACID properties, isolation levels, indexing (B-Tree vs Hash), normalization vs denormalization, query execution plans, sharding vs partitioning.
- **MongoDB / NoSQL**: Schema design schema-less vs strict, CAP theorem consistency tradeoffs, aggregation pipeline.
- **Redis**: Caching strategies (Write-through, Cache-aside), eviction policies (LRU, LFU), pub/sub.
