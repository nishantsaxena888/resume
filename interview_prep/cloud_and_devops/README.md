# Cloud & DevOps

## Focus areas for interviews:
- **AWS**: Lambda (cold starts, execution context), Step Functions (state machine orchestration), ECS (Fargate vs EC2), S3 (presigned URLs, lifecycle policies), DynamoDB (partition keys, GSIs, querying vs scanning).
- **Docker & Kubernetes**: Dockerfile best practices, multi-stage builds, basics of Kubernetes pods, deployments, services, and ingress.
- **CI/CD**: GitHub Actions workflows, secrets management, deployment strategies (blue-green, canary).
- **Infrastructure as Code**: Terraform state management, modules, plan vs apply.
