# Search & Event-Driven Architecture

## Focus areas for interviews:
- **Kafka**: Topics, partitions, consumer groups, offset management, exactly-once semantics, handling backpressure.
- **Elasticsearch**: Inverted index, analyzers, querying vs filtering, clustering, document routing, dense vectors for semantic search.
- **RabbitMQ vs Kafka**: When to use message queues (RabbitMQ/SQS) vs append-only event streams (Kafka/Kinesis).
