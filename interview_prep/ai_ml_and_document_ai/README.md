# AI, ML & Document AI

## Focus areas for interviews:
- **Document AI pipelines**: Challenges in OCR, AWS Textract integration, bounding boxes, confidence scores, combining basic OCR with heuristics to extract fields.
- **Generative AI & LLMs**: RAG (Retrieval-Augmented Generation) architecture, chunking strategies for embeddings, vector databases, prompt engineering techniques (few-shot, chain of thought).
- **Machine Learning Foundations**: Precision vs Recall, F1 score, basics of model validation, classification vs regression.
