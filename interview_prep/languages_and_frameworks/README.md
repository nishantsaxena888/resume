# Languages & Frameworks

## Focus areas for interviews:
- **Python**: 
  - GIL (Global Interpreter Lock), asynchronous programming (`asyncio` vs threading vs multiprocessing).
  - Memory management, decorators, generators, list comprehensions.
  - Python frameworks like Django/FastAPI (ORM optimization, dependency injection, Pydantic).

- **TypeScript**: 
  - Interfaces vs Types, Generics, Enums, Utility Types (Partial, Omit, Pick).
  - Type narrowing, strict null checks, 'any' vs 'unknown'.
  - Using TypeScript within React applications (typing props, state, event handlers).

- **React.js & Next.js**: 
  - Component lifecycle, Hooks (`useState`, `useEffect`, `useMemo`, `useCallback`, custom hooks).
  - State management (Context API, Redux/Zustand).
  - Next.js: Server-Side Rendering (SSR) vs Static Site Generation (SSG), App Router vs Pages Router.

- **Tailwind CSS**: 
  - Utility-first CSS methodology, JIT compiler advantages.
  - Creating custom themes, extending configuration (`tailwind.config.js`).
  - Handling responsive design classes (`sm:`, `md:`, `lg:`) and `dark:` mode.
  - Reusing styles effectively without cluttering markup (`@apply` vs extracting UI components).

- **System Design (API & Integration)**: 
  - How to build RESTful & GraphQL APIs, rate limiting, pagination, idempotency.
