# Languages & Frameworks

## Focus areas for interviews:
- **Python**: GIL, asynchronous programming (asyncio vs threading vs multiprocessing), memory management, decorators, generators.
- **React.js & Next.js**: Component lifecycle, hooks (useState, useEffect, useMemo, useCallback), Context API vs Redux, Server-Side Rendering (SSR) vs Static Site Generation (SSG), React 19 features.
- **Django/FastAPI**: Django ORM optimization (select_related, prefetch_related), middleware, FastAPI dependency injection, Pydantic validation, sync vs async endpoints.
- **System Design**: How to build RESTful & GraphQL APIs, rate limiting, pagination, idempotency.
